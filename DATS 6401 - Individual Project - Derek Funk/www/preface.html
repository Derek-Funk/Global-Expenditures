DATS 6401 | Visualization of Complex Data | Individual Project

This is the first project in the George Washington University's Visualization of Complex Data, led by Dr. Nima Zahadat. This assignment utilizes a variety of contemporary technologies to visually examine some of the top nations' spending on education, healthcare, and the military, as well as how they relate to the nations' overall spending, GDP, and population.

Specific details on what data was used for this project can be read in the Data Sources tab.

Technologies Used

Here is a summary of the various tools summoned upon in this project:

  1. HTML/CSS/JS: These are used for a lot of the word processing and displays. This Preface section, for example, is made of HTML & CSS.
  2. Google Chart API: This API is heavily relied upon for all the data visualizations. You can find extensive documentation on it here: https://developers.google.com/chart/
  3. R/Shiny:
    • R is used for all the pre-processing that is done to get raw data sources into formats that can be easily read by the visualizations created from the Google Chart API.
    • Shiny is an R package that allows you to perform front-end web programming purely with the R language. Essentially, the Shiny framework is a library of R commands that translate directly to HTML, CSS, and JavaScript. The navigation pane to the left, for example, was created with Shiny (try clicking on the 3-line icon to the upper-left). Documentation on this framework can be found here: https://shiny.rstudio.com/
